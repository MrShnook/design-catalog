#### [Link to section](https://ui.aceternity.com/components/canvas-text\#props) Props

| Prop | Type | Default | Description | Required |
| --- | --- | --- | --- | --- |
| `text` | `string` | - | The text to display with animated background. | Yes |
| `className` | `string` | `""` | Additional CSS classes for the text element. | No |
| `backgroundClassName` | `string` | `"bg-white dark:bg-neutral-950"` | Tailwind classes to determine background color for dark mode. | No |
| `colors` | `string[]` | `["#ff6b6b", "#4ecdc4", "#45b7d1", "#96ceb4", "#ffeaa7", ...]` | Array of colors for the animated lines. Supports CSS variables. | No |
| `animationDuration` | `number` | `5` | Duration in seconds for one complete animation cycle. | No |
| `lineWidth` | `number` | `1.5` | Width of the animated lines in pixels. | No |
| `lineGap` | `number` | `10` | Vertical gap between lines in pixels. | No |
| `curveIntensity` | `number` | `60` | Intensity of the bezier curve animation. | No |
| `overlay` | `boolean` | `false` | If true, positions the component absolutely for overlay use cases. | No |

#### [Link to section](https://ui.aceternity.com/components/canvas-text\#css-variable-colors) CSS Variable Colors

You can use Tailwind CSS variables for colors:

```
<CanvasText
  text="Colorful Text"
  colors={[\
    "var(--color-blue-500)",\
    "var(--color-sky-500)",\
    "var(--color-violet-500)",\
    "var(--color-teal-500)",\
  ]}
/>CopySelect Language
```

#### [Link to section](https://ui.aceternity.com/components/canvas-text\#dark-mode) Dark Mode

The component automatically adapts to dark mode by detecting the `backgroundClassName` prop. Pass your page's background color classes to ensure proper rendering:

```
<CanvasText
  text="Dark Mode Ready"
  backgroundClassName="bg-white dark:bg-neutral-950"
/>CopySelect Language
```
