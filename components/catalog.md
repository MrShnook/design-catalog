## How to Use This Catalog (Agent Guide)

### Finding the Right Component

1. **By Animation Trigger**: Use the Trigger column to find components that activate on `hover`, `scroll`, `click`, `drag`, `continuous`, or `load`
2. **By Category**: Browse sections above for your use case
3. **By Effect**: Search for keywords in Animation column (e.g., 'tilt', 'sparkle', 'gradient', 'typewriter')

### Reading Detail Files

Each `details/{slug}.json` contains:
- `animation_description` — detailed description of the motion/effect
- `code_snippet` — install command and key props
- `tech_stack` — dependencies (Framer Motion, Three.js, etc.)
- `use_cases` — suggested contexts
- `install_command` — `npx shadcn@latest add @aceternity/{slug}`

### Installation Pattern

```bash
npx shadcn@latest add @aceternity/{component-slug}
```

Most components require **Framer Motion** + **Tailwind CSS**.
Some 3D/globe components also need **Three.js** (`@react-three/fiber`, `@react-three/drei`).
