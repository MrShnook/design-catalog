## Agent Usage Guide

### Finding the Right Component
1. **By Trigger**: Filter by `hover`, `scroll`, `click`, `drag`, `continuous`, `load`
2. **By Category**: Browse sections above
3. **By Effect**: Search keywords in Animation Description (e.g., 'tilt', 'sparkle', 'typewriter')

### Detail File Schema (`details/{slug}.json`)
- `animation_description` — detailed motion/effect description (THE key field)
- `animation_type` — trigger: `hover` | `scroll` | `load` | `click` | `drag` | `continuous`
- `category` — component category
- `code_snippet` — install command and key props
- `tech_stack` — e.g. `['Framer Motion', 'Tailwind CSS']`
- `install_command` — `npx shadcn@latest add @aceternity/{slug}`
- `use_cases` — suggested contexts
- `free` — `true` for /components/, `false` for /blocks/

### Standard Install
```bash
npx shadcn@latest add @aceternity/{component-slug}
```
Most components: **Framer Motion** + **Tailwind CSS**.
Globe/3D: also needs **Three.js** (`@react-three/fiber`, `@react-three/drei`).
