### Story

Outfit is the official typeface for brand automation company [outfit.io](https://outfit.io/). Inspired by the ligature-rich outfit wordmark, Outfit.io is delighted to release it's own type family. The Outfit typeface links the Outfit written voice to Outfit product marks; on brand by default.

The proof files and QA tests are available [automatically via GitHub Actions](https://outfitio.github.io/Outfit-Fonts/)

Outfit is licensed under the SIL Open Font License. You can find more details about this project [here](https://github.com/Outfitio/Outfit-Fonts).
