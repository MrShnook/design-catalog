### Story

Designed to integrate with the character of the Mozilla FirefoxOS, the Fira typefaces also aim to cover the legibility needs for a large range of handsets varying in screen quality and rendering.

This project is led by [Carrois](https://carrois.com/), a type foundry based in Berlin.

Fira Sans is licensed under the SIL Open Font License. You can find more details about this project [here](https://github.com/mozilla/Fira).
